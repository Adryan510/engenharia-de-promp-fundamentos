# engenharia-de-prompt-fundamentos

## Prompt Genérico

Pergunta:
O que é Machine Learning?

Resposta:
Machine Learning é uma área da inteligência artificial que permite que sistemas aprendam a partir de dados e melhorem suas decisões sem serem programados diretamente para cada tarefa.

---

## Prompt Estruturado

Prompt:
Atue como um professor. Explique o que é Machine Learning para um iniciante usando uma analogia com esportes em dois parágrafos.

Resposta:
Machine Learning pode ser comparado ao treinamento de um atleta. No início, o atleta pratica várias vezes e aprende com seus erros. Da mesma forma, um algoritmo analisa muitos dados para aprender padrões e melhorar suas previsões.

Assim como um jogador melhora com treino e experiência, o algoritmo também melhora à medida que recebe mais dados e feedback, tornando suas decisões cada vez mais precisas.

---

## Análise Crítica

A resposta do prompt estruturado foi mais útil porque apresentou contexto, objetivo e formato claros. Isso reduziu a ambiguidade da pergunta e tornou a explicação mais organizada e fácil de entender para iniciantes.
